ผักผลไม้ 3 สีนี้..กินดี ต้านโรค!

อย่างที่รู้ๆกันอยู่แล้วว่า เจ้าผักและผลไม้นั้นเป็นพืชที่มากด้วยคุณค่า สามารถต่อต้านโรคภัยได้สารพัด และนี่ก็คือผักผลไม้ 3 สีที่มีคุณประโยชน์ต่อร่างกายอย่างที่หลายๆคนอาจะไม่เคยรู้!

 

 

1. ผักผลไม้สีขาว
  ขิง : มีจินเจอรอล ซึ่งเป็นสารพฤกษเคมี ที่ช่วยลดการอักเสบ ช่วยบรรเทาอาการปวดเมื่อยกล้ามเนื้อหลังจากออกกำลังกาย
  หอมใหญ : อุดมไปด้วยเคอร์ซิทิน ซึ่งเป็นสารประกอบที่ทําให้อึดขึ้น แนะนําให้กินสัปดาห์ละ 5 ครั้ง
  มันฝรั่ง : มีกรดไฮดรอกซีซินนามิก ช่วยต้านอนุมูลอิสระ กินมันฝรั่งบด 1 ถ้วย จะช่วยปรับปรุงความจําทั้งระยะสั้นและระยะยาวให้ดีขึ้น
  ดอกแค : ยอดแคมีเบต้าแคโรทีนสูงมาก ดอกแคมีวิตามินซี ช่วยแก้หวัดได้ดี แถมยังทำให้ผิวสวยอีกต่างหาก
  กล้วย : อุดมไปด้วยน้ำตาลธรรมชาติที่ช่วยเสริมเพิ่มพลังงานให้กับร่างกายแบบทันทีทันใด

2. ผักผลไม้สีเขียว
  ผักโขม : ผักโขมสุก 1 ถ้วย ให้ปริมาณสารต้านอนุมูลอิสระ ช่วยบำรุงจอประสาทตาได้ดี
  เมล่อน ฮันนี่ดิว : มีส่วนผสมของโพแทสเซียม ช่วยลดผลข้างเคียงจากการกินอาหารฝรั่งที่มีโซเดียมสูง จึงช่วยรักษาระดับความดันเลือด
  แนะนําให้กิน 4 ชิ้น/สัปดาห์
  อะโวคาโด : อุดมไปด้วยวิตามินอี และแอนติออกซิแดนต์ ช่วยบํารุงผิวและลดระดับคอเลสเตอรอลได้ 1 ถ้วย จะช่วยปรับปรุงความจําทั้ง
  ระยะสั้นและระยะยาวให้ดีขึ้น
  บรอกโคลี : มีเบต้าแคโรทีน เส้นใยอาหารวิตามินซี ซึ่งมีคุณสมบัติในการต่อต้านมะเร็ง และช่วยขจัดกลิ่นเหม็นต่างๆในตัวได้ด้วย
  ขึ้นฉ่าย : กินทั้งต้นจะช่วยลดความดันโลหิตและรักษาโรคนิ่วได้ค่ะ

3. ผักผลไม้สีส้ม/เหลือง
  สับปะรด : มีสรรพคุณช่วยย่อยอาหารจําพวกเนื้อสัตว์ ช่วยควบคุมน้ำตาลในเลือด และช่วยลดความเสี่ยงของมะเร็ง
  ฟักทอง : มีเส้นใยสูง ให้พลังงานต่ำ ไขมันน้อย เหมาะกับคนที่ต้องการควบคุมน้ำหนัก
  อะโวคาโด : อุดมไปด้วยวิตามินอี และแอนติออกซิแดนต์ ช่วยบํารุงผิวและลดระดับคอเลสเตอรอลได้ 1 ถ้วย จะช่วยปรับปรุงความจําทั้ง
  ระยะสั้นและระยะยาวให้ดีขึ้น
  แครอท : อุดมไปด้วยเบต้าแคโรทีน ช่วยป้องกันการเกิดโรคมะเร็ง บำรุงสายตา ผิวพรรณ ระบบประสาท ควรปรุงให้สุกก่อนกิน เพื่อให้ได้
  คุณค่าทางอาหารสูงสุด
  มะละกอห่ามๆ : หากกินนานถึง 2 ปี อาจจะช่วยเปลี่ยนสีผิวหน้าที่เป็นฝ้าให้หายได้ โดยไม่ต้องพึ่งครีมทาฝ้าเลยล่ะ
